# Jump Calculator

A fun web calculator that jumps when you press the '=' key! This calculator has a unique jump animation effect that triggers every time you calculate a result.

## Features

- Standard calculator functionality (addition, subtraction, multiplication, division)
- Jump animation effect when pressing '=' button or Enter key
- Responsive design that works on mobile and desktop
- Keyboard support for all operations
- Modern UI with gradient background and glass-morphism effects

## Technical Details

- Built with HTML, CSS, and JavaScript
- Uses CSS animations for the jump effect
- Fully responsive design
- Keyboard support for all calculator operations
- Modern UI with glass-morphism effects

## Keyboard Shortcuts

- `0-9`: Number inputs
- `+`, `-`, `*`, `/`: Operators
- `Enter` or `=`: Calculate and trigger jump animation
- `Escape`: Clear the calculator

## Files

- `index.html`: Main HTML structure
- `style.css`: Styling and animations
- `script.js`: Calculator logic and functionality
